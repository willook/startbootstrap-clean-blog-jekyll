이전까지의 주제가 마무리되고 이제는 새로운 주제를 가지고 프로젝트를 진행하려고 합니다. 새롭게 프로젝트를 시작하기전 지금까지의 프로젝트 내용을 간략하게 정리하면 다음과 같습니다.

- Using ROS and ROSARIA to move robot
A set of software libraries and tools that help you build robot applications

- Moving robot from one GPS point to another
Raspberry pi GPS module was used.
It moved with a margin of error of about 5%.

- Communication between Commander and Robot.
The Python socket module was used.

간략하게 영어로 지금까지 했던 프로젝트에 대해 적어보았습니다. 로봇을 움직일 수 있게 하는 것이 첫 번째 목표였습니다. 왜냐하면 움직일 수 있어야 이후의 프로젝트의 진행에 있어서 이전에 받았던 시나리오대로 할 수 있었기 때문입니다. 다음으로는 앞으로의 프로젝트에 대해서 설명을 하겠습니다. 다음 프로젝트는 로봇이 목표로 지정한 곳에 도착한 이후의 시나리오에 대해서 다루고 있습니다.



- Goal of our project

Our project does not end with robots moving to GPS points and searching for troops.
The goal is to find the distance to the army
and get the GPS coordinates of the corresponding location.


위와 같은 내용의 다음 프로젝트를 구상하였습니다. 이제는 앞으로의 프로젝트에 필요한 기술과 장비들을 목요일까지 생각해보고 조사까지 끝마칠 생각입니다. 가장 먼저 생각했던 것은 Image segmentation과 stereo camera를 이용한 대상까지의 거리 측정입니다. 오늘의 task는 환경 설정을 위주로 진행하였습니다. Image segmentation을 위한 딥러닝 환경 세팅과 stereo camera를 구성하기 위한 장비 검색, stereo camera를 이용하여 Image depth 추출 방법등에 대한 조사를 진행하는 것으로 오늘의 task는 마무리 되었습니다.
